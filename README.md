# Yushi Marketing Technology — Live Streaming

Sales page and pitch deck for Yushi Marketing Technology's **Live Streaming & Production** service (LiveVibe by Yushi).

## Contents

- **`index.html`** — Scrolling sales / landing page (hero, portfolio galleries, services, rate card, contact).
- **`deck/index.html`** — 11-slide 16:9 pitch deck (business profile). Navigate with ← → arrow keys, on-screen arrows, or the dot indicator; press **F** for fullscreen.

Both are self-contained static HTML (no build step) — open directly or serve any static host.

**Production:** https://livestream.yushi-marketing.com/ (sales page) · https://livestream.yushi-marketing.com/deck/ (pitch deck)
